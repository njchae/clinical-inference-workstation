# Limitations

- Fully synthetic data means the visual task is engineered for demonstration, not real-world generalization.
- The exported metrics describe performance on synthetic held-out data only.
- The workstation UI is intentionally thin and does not attempt to recreate a full product.
- Authentication, audit persistence, and Azure-ready deployment assets are included only as public-safe reference implementations.
- The OpenAI and Anthropic labeling adapters are represented as provider shapes; CI uses the local structured-label provider rather than live third-party calls.
- This repo demonstrates methodology and architecture, not clinical validity.
